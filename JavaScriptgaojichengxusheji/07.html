必包

必包几乎是每个前端面试基本都会问到的..
我理解的必包 就是作用域的延长。 首先 作用域就类似于下面这玩意儿 

	
	function a(b)
	{
		var c = '';
	}

	a('a');

传入的参数，会当做a 里面的私有变量 复制一份传进去

然后，当方法执行完以后，b&c 都会被销毁

你是无法访问ta的 

	function a(b)
	{
		var c = "";

		return function() {
			return c;
		}
	}

接下来 你就可以通过 返回的方法 访问到c

ta本来该被销毁 作用域延长了。 可以当做一个私有的变量、

坏处。就是内存问题, 太多必包对象。可能会造成内存问题
